Collaboration Over GitHub: Admin Module

 

You must enter the following details for configuring a project

 
                                      Project Name:

                                      Owner Name:

                                      Enter GitHub Repo URL: (e.g.: https://github.com/ritubits/MathTutorialProject.git)

                                      Enter Temporary Folder Path: (e.g.: D:\TestGitProjectRepo)

                                      Enter Source Folder Path: (e.g.: D:\TestGitProjectRepo\MathProject\src\twoDShapes)
